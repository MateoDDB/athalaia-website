Athalaïa, logiciel automatisé de pentest

Scan du réseau

Athalaïa scan le réseau, il repère les machines présentes et récupère leurs adresses IP. Des informations supplémentaires comme le type de machine ou d'OS est détecté.

Scan de l'OS et version

Athalaïa effectue un scan sur la ou les machines choisis entré dans le champ des cibles. Une fois scanné Athalaïa récupère les types de logiciels utilisés et leur versions. Les données sont stockés et traité.

Scan de vulnérabilité

Athalaïa compare les logiciels et versions de la machine scanné par rapport à sa base de données intégré. Une fois comparé, Athalaïa détermine les corespondances entre les logiciels et versions présentent aux CVE présente dans la base de données.

Rapport de failles de sécurité

Athalaïa stock les CVE encontrés et crée un rapport détaillé des failles de sécurité associé aux ports ouverts. Un descriptif des failles de sécurité et correction proposé. Pour aller plus loin, un site web de correction de CVE est affiché.

Combien de temps prend un test de sécurité Athalaïa ?

Athalaïa prend au maximum 2h de tests au sein d'une entreprise. Pour cela, le temps de rédaction du rapport est fortement réduis. Athalaïa peut scanner directement tout une infrastructure réseau en un clic.

Une fois le test réalisé par nos techniciens, une proposition de devis est faite pour une correction plus poussé.

Test hors connexion

Le test de sécurité est réalisé hors ligne. Le logiciel Athalaïa n'utilise pas la connexion internet pour réaliser son pentest.

Test sécurisé

Le test est sécurisé car le logiciel Athalaïa ne laisse pas de signature sur le réseau. Une signature réseau peut être identifié par les "pirates informatiques" et donc repérer les entreprises en cours de test de sécurité.
